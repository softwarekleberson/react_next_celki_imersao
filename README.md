como rodar o projeto baixado, sem o node_modules
### npm install

Rodar projeto
### npm run dev

1)criar uma pasta react_node_react_native e depois pasta SITE, em SITE digitar esse comando abaixo
### npm init 

2) instalar o react junto com o next, o next vai transformar o javascript em html, melhor para os buscadores de pesquisa
### npm install next react react-dom

3) em package.json, em script coloque isso

"dev": "next dev",
"build": "next build",
"start": "next start"

Rodar projeto
### npm run dev
